Пример игрового поля:

Игрок №1 начинает игру из нижнего левого угла.
Игрок №2 начинает игру из верхнего правого угла.

Захват игрового поля.

Клетка игрового поля может быть захвачена только тем цветом которым она залита. Т. е. Синяя клетка может быть захвачена только синим цветом, красная – красным и т. д. После захвата такой клетки происходит осмотр соприкасающихся с ней клеток и в том случае если их цвет является точно таким же происходит и их захват. После чего этот же принцип применяется и для вновь захваченной клетки. И так до тех пор пока не будут захвачены все соприкасающиеся клетки одно цвета. Соприкасающимися считаются клетки соприкасающиеся одной из сторон с захваченной клеткой.

Пример соприкасающихся клеток:

Принцип захвата является принципом “эпидемия” – одна клетка “заражает” другую, та – “третью” и т. д. После захвата клетка в дальнейшем не приносит очков и может быть захвачена любым цветом. Цвет, которым произошел захват, считается недоступным для другого игрока в его ход. Т. е., если Игрок №1 выбрал для захвата красный цвет , то по окончании его хода Игрок №2 сможет использовать для захвата любой цвет кроме красного. И наоборот. В результате ни один игрок никогда не сможет захватить хоть часть поля уже захваченного другим игроком.

Выбор цвета для захвата игрового поля происходит путем нажатия на любую из доступных кнопок данного цвета.

Пример выбора цвета:

После выбора цвета происходит захват игрового поля начиная со стартовой позиции для каждого игрока. Для Игрока №1 это нижний левый угол игрового поля, для Игрока №2 это верхний правый угол игрового поля. Каждая захваченная клетка приносит определенное количество очков (см. таблицу). После того как один из игроков захватит половину или больше игрового поля игра заканчивается. По окончании игры происходит процентный подсчет захваченных клеток каждого цвета. Игрок захвативший клеток какого-либо цвета больше чем другой игрок получает дополнительные очки. После подсчета разница в очках записывается на счет победителя (если у него очков больше чем у побежденного).
